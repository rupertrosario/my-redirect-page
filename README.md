# Angel One Redirect Page

**Category:** Trading / Broker API Utility  
**Purpose:** Redirect page used for Angel One SmartAPI app registration.

## Scope

This repository should contain only the redirect-page files and minimal documentation required for SmartAPI registration or OAuth redirect testing.

## Suggested structure

```text
public/         Static redirect page assets
docs/           Setup notes and registration steps
```

## Cleanup rules

- Do not commit API keys, client secrets, access tokens, or refresh tokens.
- Keep broker-registration notes sanitized.
- Document the configured redirect URL and deployment target without exposing credentials.
